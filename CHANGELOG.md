# Changelog

本文档记录 ClassDrive 的版本更新历史和重要变更。

## [Unreleased]

### Added - 新增功能

#### 老师资料分享功能
- **教师端分享管理**：
  - 在老师资料空间中可将文件或文件夹生成分享链接
  - 支持设置访问权限：仅查看（只能预览）或允许下载
  - 支持设置有效期：1天、7天、30天、永久或自定义日期
  - 支持安全码保护：访客需输入正确安全码才能访问
  - 创建分享后显示分享链接和安全码，支持一键复制
  - 新增"分享管理"界面，可查看所有分享记录（状态、访问次数、创建时间等）
  - 支持编辑分享权限和有效期
  - 支持停用/启用分享（不删除记录）
  - 支持删除分享
  - 支持重置安全码

- **访客端分享访问**：
  - 通过分享链接免登录访问（路由：`/share/:token`）
  - 需要安全码时输入验证后访问
  - 文件分享：直接预览文件，根据权限显示下载按钮
  - 文件夹分享：浏览目录结构、进入子目录、预览和下载文件
  - 权限控制：仅查看权限时不显示下载和打包下载按钮
  - 过期、停用或源文件已删除的分享自动拒绝访问

- **系统设置**：
  - 新增"默认分享有效期"设置（范围 1-3650 天，默认 7 天）
  - 创建分享时自动应用该默认值

- **操作审计**：
  - 记录教师分享操作：创建分享、修改分享、删除分享、重置安全码
  - 记录访客访问操作：查看分享、预览文件、下载文件、下载文件夹压缩包
  - 在统一操作日志中查看完整审计记录

### Technical Details - 技术细节

- **后端**：
  - 新增 `library_shares` 表存储分享记录
  - 新增 `/api/library-shares` API 系列（教师端 CRUD）
  - 新增 `/api/share/:token` API 系列（访客端访问）
  - 安全码使用 bcrypt 加密存储
  - 分享会话通过 HttpOnly Cookie 管理
  - 目录遍历越权防护
  - 完整的单元测试覆盖

- **前端**：
  - 新增 `library-shares` Pinia store
  - 新增 ShareCreateDialog、ShareManageDialog、ShareEditDialog 组件
  - 新增 ShareView 页面（访客端）
  - FilesView 集成分享入口（仅 library 空间）
  - 完整的 Vitest 单元测试

### Security - 安全性

- 分享 token 使用 16 字节随机生成（base64url 编码）
- 安全码使用 bcrypt (cost 10) 加密存储
- 分享会话通过 HttpOnly、SameSite Cookie 防护
- 严格的权限校验：仅 library 空间可创建分享
- 路径遍历防护：访客无法通过 `../` 访问分享范围外的文件
- 过期、停用、源文件删除自动拒绝访问
