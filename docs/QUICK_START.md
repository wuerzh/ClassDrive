# ClassDrive 快速开始指南

**版本**: v1.5  
**更新日期**: 2026-06-13

---

## 📖 概述

ClassDrive 是一个轻量级的课堂教学文件与作业管理系统，专为 50 人以下班级的局域网教学环境设计。

**核心功能**:
- 📁 文件管理（上传、下载、预览、编辑）
- 📝 作业管理（发布、提交、批改、统计）
- 👥 学生管理（导入、编辑、权限控制）
- 🔗 文件分享（限时、限次、密码保护）
- 📊 审计日志（操作记录、登录日志）

## 🚀 快速部署（5 分钟）

### 方式一：使用预编译二进制（推荐）

1. **下载发布包**:
   ```bash
   # 下载最新版本
   wget https://github.com/your-org/classdrive/releases/latest/classdrive
   chmod +x classdrive
   ```

2. **启动服务**:
   ```bash
   ./classdrive
   ```

3. **访问系统**:
   - 打开浏览器访问: `http://localhost:8050`
   - 默认管理员账号: `admin` / `admin123`
   - **首次登录后请立即修改密码**

### 方式二：从源码编译

**前置要求**:
- Go 1.21+
- Node.js 18+
- npm 或 pnpm

**步骤**:
```bash
# 1. 克隆仓库
git clone https://github.com/your-org/classdrive.git
cd classdrive

# 2. 构建前端
cd frontend
npm install
npm run build
cd ..

# 3. 构建后端
go build ./cmd/classdrive

# 4. 运行
./classdrive
```

## 🎯 首次使用

### 1. 登录系统
- 访问: `http://localhost:8050`
- 账号: `admin`
- 密码: `admin123`

### 2. 修改密码
- 点击右上角头像 → 设置
- 修改管理员密码
- **强烈建议使用强密码**

### 3. 创建班级
- 导航到"班级"页面
- 点击"新建班级"
- 输入班级名称（如"高一1班"）

### 4. 添加学生
两种方式：
- **手动添加**: 点击"新增学生"，逐个输入
- **批量导入**: 准备 CSV 文件，点击"导入学生"

CSV 格式示例：
```csv
学号,姓名
2024001,张三
2024002,李四
```

### 5. 上传文件
- 进入"文件"页面
- 拖拽文件到页面，或点击"上传文件"
- 支持文件夹上传

### 6. 发布作业
- 进入"作业"页面
- 点击"新建作业"
- 填写作业信息（标题、描述、截止时间）
- 选择班级

## 🔧 常见问题

### Q1: 忘记管理员密码怎么办？
**A**: 
```bash
# 停止服务，删除数据库
rm var/data/classdrive.db

# 重新启动，系统自动重建（恢复默认密码）
./classdrive
```

### Q2: 如何更改端口？
**A**: 使用环境变量
```bash
PORT=9000 ./classdrive
```

### Q3: 文件上传大小限制？
**A**: 默认 500MB，可通过环境变量修改
```bash
MAX_UPLOAD_SIZE=1073741824 ./classdrive  # 1GB
```

### Q4: 如何备份数据？
**A**: 备份以下目录
```bash
# 数据库
cp -r var/data var/data-backup-$(date +%Y%m%d)

# 文件存储
cp -r var/storage var/storage-backup-$(date +%Y%m%d)
```

### Q5: 支持哪些文件预览？
**A**: 
- **在线预览**: 图片、PDF、文本文件、Markdown
- **在线编辑**: 文本文件 (.txt, .md, .js, .py, .go 等)
- **其他文件**: 下载后查看

### Q6: 性能优化建议？
**A**: 
- 定期清理旧作业和过期分享
- 避免单个文件夹超过 1000 个文件
- 使用 SSD 存储
- 本次更新（v1.1.0）已优化大列表性能
