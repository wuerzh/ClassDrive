## 🔧 如果遇到问题

### 端口被占用
```bash
# 方法1：修改端口
set CLASSDRIVE_PORT=8080
./classdrive.exe

# 方法2：以管理员身份运行
```

### 分享链接无法访问
1. 打开浏览器开发者工具 (F12)
2. 查看 Console 标签的错误
3. 查看 Network 标签的请求状态
4. 查看服务器控制台日志
